    \noindent And now for something completely different: parametric equations. Why? The best way to understand is to look at something in motion:\\[1ex]

    \noindent We can describe motion in a coordinate plane, but we cannot also show the time of motion. To do that, we introduce a third variable, $\displaystyle t$, called a parameter. Bad news: this is challenging. Good news: we can use our calculator.\\[1ex]

    \noindent Describe the graph of the relation determined by $\displaystyle x = \sqrt{t},\ y = t,\ t \ge 0$. Indicate the direction in which the curve is traced and find the Cartesian equation.

    \emptygraph

    \noindent Notice how important domain is\dots we call the domain $\displaystyle I$ the parameter interval.\\[1ex]

    \noindent Definition: If $\displaystyle x$ and $\displaystyle y$ are given as functions $\displaystyle x = f(t),\ y = g(t)$ over an interval of $\displaystyle t$-values, then the set of points $\displaystyle (x,y) = (f(t), g(t))$ defined by these equations is a parametric curve. The equations are parametric equations for the curve.\\[2ex]

    \noindent Also notice that direction matters\dots the parameter $\displaystyle t$ determines where the curve starts and ends. Describe the graph of the relation determined by the following. Determine initial and terminal points and find the Cartesian equation.

    \[x = 2\cos t,\quad y = 2\sin t,\quad 0 \le t \le 2\pi\]\\[2ex]

    \newpage

    \noindent Graph the parametric curve: $\displaystyle x = 3\cos t,\quad y = 4\sin t,\quad 0 \le t \le 2\pi$

    \emptygraph

    \vspace{2ex}

    \noindent At this point, you might think that parametric equations are used primarily for graphs of conics. However, we can also graph a line segment. Draw and identify the graph of the parametric curve: $\displaystyle x = 3t,\ y=2-2t,\ 0 \le t \le 1$

    \emptygraph

    \noindent Find a parameterization for the line segment with endpoints $\displaystyle (-2,1)$ and $\displaystyle (3,5)$.
